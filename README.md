# TeaFlow

TeaFlow is a dependency-free static web app for timing tea, calculating tea-to-water ratios, tracking loose-leaf re-steeps, saving custom brew profiles, and tuning recipes from taste notes.

The visual theme uses a peaceful generated nature background inspired by misty Chinese mountain scenery and Japanese garden elements.

## Features

- Tea timer presets for green, black, oolong, white, and herbal tea.
- Custom brew profiles with tea name, grams, water amount, temperature, steep time, and rating.
- Brew calculator using 2-3 g of tea per 250 ml water.
- Loose-leaf re-steep tracker for 60, 75, and 90 second steeps.
- Taste-note suggestions for bitter, weak, grassy, strong, and perfect brews.

## Run

Open `index.html` in a browser.
